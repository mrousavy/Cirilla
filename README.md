# Commands
These are the commands you can use:


**BotInfo**
```
$info
```
**Clean**
```
$clean
$clean count
```
**Connection**
```
$ping
```
**Google**
```
$google query
```
**Greetings**
```
$hi
$hello
$hallo
$seas
$servas
$sup
```
**Help**
```
$help
$help command
```
**Link**
```
$links
$addlink link
```
**Maths**
```
$square num
$sqrt num
$calc expressions
$pi
```
**Randoms**
```
$flip
$random maximum
```
**Reddit**
```
$dankmeme
$reddit rsubreddit
```
**UserInfo**
```
$stats user
$stats
```
**Votekick**
```
$votekick user
```
**Xp**
```
$xp user, xp
$xp
$xp user
```
